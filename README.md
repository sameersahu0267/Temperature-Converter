# Temperature-Converter
A simple and responsive Temperature Converter built using HTML, CSS, and JavaScript.  
This application allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin.

## Live Demo
https://sameersahu0267.github.io/Temperature-Converter/

## Features
- Convert temperature between Celsius, Fahrenheit, and Kelvin
- Clean and modern user interface
- Input validation for correct values
- Responsive layout using Flexbox

## Technologies Used
- HTML5
- CSS3
- JavaScript
